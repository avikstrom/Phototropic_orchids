This code controls the up-down motion of orchid flowers controlled with an SG90 servo via string. Two photoresistors located in flower stamens provide input. 
The servo reels in or releases string based on which of the two orchids has greater light exposure. See .png for wiring schematic. The two resistors are 220 ohms.

15Mar2026: I rescued my breadboard and moved the circuit to a perfboard. Originally it did not work, but then I learned just how important it is to use a jumper and not have cold solder joints on the DIY'ed power rail.
